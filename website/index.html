What are you looking at?.

DankLyrics is a lyrics finding website, API, and Go package.

What you see is what you get.

My main intention here was to create a lyrics service for DankMuzikk, so initially I made this to test my Genius Go client Gonius and let the site public just to embrace CS1.6.css, then Genius added a bot protection thingy, so scrapping for lyrics wasn't possible anymore, I found LyricFind which was promising, but then I didn't know when will they go houdini like Genius, and I was like "I should have my own lyrics service instead of releing on providers", Ironically enough, DankLyrics still relies on providers to seed the data :), But soon enough (hopefully) DankLyrics will have a shitload of lyrics, and even more hopefully for it to be from the community submissions.

So here you have it, DankLyrics in flesh and bone, no trackers no nothing, and it's open source (check the footer), hoping to have an amazing lyrics provider.

And for the record, every API call made to api.danklyrics.com/lyrics adds the found lyrics to DankLyrics' database, so every search here, or when fetching lyrics for a song in DankMuzikk grows the database even further. And if you're bothered by this feel free to use the Go package to skip the "growning the database part", but again even tho it grows with the search, I have no idea who did the search, you can check the source code to make sure that DankLyrics doesn't snoop around your data, And I would really appreciate it if you help me have more lyrics for DankLyrics.
